# ◼ 프로젝트 개요

- 고등학교 내신 성적과 모의고사 성적을 간편하게 관리할 수는 없을까?
- 학생과 선생님이 서비스를 통해 원활하게 대입 상담, 학업 설계 지도를 할 수 없을까 ?

### ⇒  고등학교 성적을 통합적으로 관리할 수 있는 서비스를 만들어보자!

</br>


## ◼ 서비스 대상

- 대입을 준비하는 **대구광역시** 일반고・자율고 **고등학생**
- 학생들의 성적을 **체계적**으로 관리하고 싶은 고등학교 **선생님**

# ◼ HiSchool의 주요 기능

**1. 학생별 개인 성적 현황 제공** <br>
    - 학생: 주요 과목 최고 등급, 현재 등급 / 최근 응시 시험 등급<br>
    - 선생님: 최근 응시 시험 학급 성적 평균(내신, 모의고사)<br>
    - 그래프로 한 눈에 보기 가능<br>
    
 **2. 학급별 시간표 제공**: 학기별 시간표<br><br>
 **3. (학생) 급식표 제공**: 주간, 월간 급식표<br><br>
 **4. (선생님) 학사 일정 제공**: 해당 년도 학사 일정<br><br>

 **5. 내신성적 자동계산기능 제공**<br>
    - 전교석차 및 등급자동계산<br>
    - 학급석차 자동계산<br><br>
 **6. 학생기록부 관리**<br>
    - 출결기록부 및 진로상담<br><br>
 **7. 학생 성적 기간별 다운로드 기능**<br><br>
 **8. (관리자)공지사항기능 및 교원관리**<br>
